WordPress site management tools for Ohio County Sheriffs.

== Description ==
Originially developed for the Belmont County Sheriff's Office, the Ohio Sheriff Tool plugin is designed to help county sheriff's manage their WordPress based website. Tools include winter road level notifications and road closures. Future releases will include tools to manage sheriff sales and a wanted database.

For more information or to give money toward the plugin's development, <a href="http://www.10twebdesign.com/plugins/ohio-sheriff">visit the plugin's website</a>. Bug reports can be sent to <a href="mailto:brock@10twebdesign.com">brock@10twebdesign.com</a>.
